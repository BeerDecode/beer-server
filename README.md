# BeerEncoded

## Requirements

* NodeJS 7.4.0 or higher

## Install

Clone the project : 

```bash
$ git clone git@github.com:BeerDecodedProject/beer-server.git
```

Install dependencies :

```bash
$ npm install
```

## Start it

```bash
$ npm start
```

## Generate shitty datas

```bash
$ npm run shitty-datas
```
